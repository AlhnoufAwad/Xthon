# Xthon# Xthon Hackathon Website

## Overview

This project is the official website for **Xthon Hackathon**.
The website introduces the hackathon, its tracks, timeline, partners, and the organizing team.

It is designed to provide participants with clear information and an engaging user experience.

---

## Features

* Hackathon introduction section
* Event timeline
* Hackathon tracks and domains
* Partners and sponsors section
* FAQ section
* Team members page with interactive cards
* LinkedIn links for team members

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Team

The website presents the organizing team of **Xthon Hackathon**, including:

* Event Leaders
* Media Team
* Presentation Team
* Corporate Communication Team
* Marketing Team
* Creativity Team
* Hackathon Team

Each member card includes their name, role, and LinkedIn profile.

---

## Purpose

The goal of this website is to provide a clear digital platform for the hackathon and highlight the efforts of the organizing team.
